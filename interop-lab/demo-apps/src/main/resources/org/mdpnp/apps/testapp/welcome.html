OpenICE Launch Settings
Scroll down for more information
    Application
  • The ICE_Supervisor hosts clinical apps.
  • The ICE_Device_Interface uses this computer to connect a medical device to an ICE_Supervisor.
    Domain Id
  • OpenICE systems on the same network can coexist by using different integer domain identifiers. If there are no other ICE systems on your network you may use any value.
    Internal Patients
  • By default a list of patients is populated from https://fhir.openice.info. Check this box to skip this connection and use a local list of patients.
